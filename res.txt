    \textbf{BreathPracticesController}
    \newline

    Метод BreathPractices возвращает 
    представление BreathPractices.cshtml 
    в котором выводиться описание дыхательных практик из таблицы BreathPractices
    \newline

    Метод CreateBreathPractices
    возвращает представление CreateBreathPractices.cshtml
    в котором выводиться форма для добавления новой дыхательной практики. 
    \newline

    Метод UpdateBreathPractices 
    возвращает представление UpdateBreathPractices.cshtml
    в котором выводиться форма для редактирования существующей дыхательной практики. 
    Входной параметр id дыхательной практики.
    \newline

    Метод CreateBreathPracticesPOST записывает новую дыхательную практику в базу. 
    Входной параметр BreathPractice practice. 
    При создании новой дыхательной практики отправляется пуш уведомление
    \newline

    Метод UpdateBreathPracticesPOST перезаписывает 
    существующую дыхательную практику в базу. 
    Входной параметр объект BreathPractice practice.
    \newline

    Метод DeleteBreathPractices.Удаляет дыхательную практику из базы. Входной параметр 
    int id.
    \newline
